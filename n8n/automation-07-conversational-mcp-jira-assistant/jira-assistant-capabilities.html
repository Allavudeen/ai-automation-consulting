Jira Assistant — What it can do

A quick reference for the conversational Jira interface

This assistant lets you manage Jira issues in plain English — no project keys, transition IDs, or JQL to memorize. Type what you want the way you'd say it to a teammate, and it translates that into the correct Jira action. Below is what it currently supports, with real example phrasings.

Core capabilities

Look up an issue

Pulls full details on any issue — status, assignee, comments, and available next steps — in a single request.

What's the status of PDPT-18?

Search across issues

Finds issues matching a description — by project, status, type, or a mix — without writing any query syntax.

Show me everything still in To Do on the PDPT project

Create an issue

Turns a rough description into a properly structured Bug, Task, or Story — with a clear title, expanded description, and acceptance criteria when relevant.

Create a bug: login page fails on Safari

Update issue fields

Edits a title, description, or other fields on an existing issue.

Update PDPT-19's summary to "Login Page Failing on Safari - Confirmed Repro"

Assign an issue

Reassigns an issue to a teammate. Needs their exact email or Jira username — first names alone can't be safely matched.

Assign PDPT-14 to priya@company.com

Add a comment

Adds a comment to an issue's activity log — useful for status notes or context for the next person.

Add a comment on PDPT-18: waiting on QA confirmation before closing

Move an issue's status

Transitions an issue to a new status — the assistant checks which transitions are actually valid for that issue before moving it, so it won't guess.

Move PDPT-18 to In Progress

List your projects

Shows every Jira project you have access to, with its key — a good starting point if you're not sure what's available.

What Jira projects do I have access to?

Growing on demand

~200 more capabilities available across Jira & Confluence

This version covers the actions your team uses most day-to-day. Beyond these, the underlying platform supports roughly 200 additional Jira and Confluence capabilities — sprint management, bulk operations, worklogs, watchers, issue linking, Confluence page creation and search, and more — any of which can be enabled on request as your workflow needs grow.

Currently active in this version:

  • Issue lookup, search, creation, updates, assignment
  • Comments and status transitions
  • Project visibility
This assistant makes real changes to your Jira workspace. It will ask for confirmation before anything ambiguous or destructive — but for critical changes, it's good practice to verify the result directly in Jira.
